# Nmap SOC Lab

This repository contains structured Nmap scans and security analysis performed in an **authorized internal lab environment**.  
It is designed to demonstrate **SOC Level 1 analyst skills** including network discovery, port scanning, service enumeration, and security impact analysis.

All scans are documented with a **defensive security mindset**, focusing on visibility, exposure assessment, and incident detection support.

---

## Objectives

- Perform network and host discovery
- Identify exposed TCP and UDP services
- Detect service versions and operating systems
- Analyze web service security posture
- Practice SOC-style documentation and reporting

---

## Lab Environment

- Authorized internal network
- Private IP address space (RFC1918)
- Nmap version: 7.9x
- Defensive, SOC-focused use cases only

> No unauthorized or illegal scanning was performed.
